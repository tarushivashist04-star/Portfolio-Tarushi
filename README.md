# Portfolio-Tarushi
A  portfolio built with React, HTML, CSS, and JavaScript. It showcases my skills, projects, and contact information with responsive design, animations, and a magical UI experience.

🌟 Personal Portfolio Website:

A modern Harry Potter themed portfolio built using React (Vite) to showcase my projects, skills, and journey as a Frontend Developer and Computer Science student (AI & ML).
This portfolio was built from scratch to practice real React component structure, responsive design, and modern frontend development techniques, rather than relying on templates.

📖 About the Project:

I created this portfolio to represent my learning journey in frontend development and to have a personal platform where I can showcase my projects and skills.
Instead of using ready-made designs, I focused on building everything myself to better understand React components, layout design, styling, and animations.
The goal of this project was to build a clean, interactive, and visually engaging portfolio with smooth navigation and a magical UI inspired by the Harry Potter theme.
